# DUORECALL (多邻记)

## 1. 多邻记是什么？
多邻记是一个利用ai人工智能辅助学习复习记忆多邻国句子卡片的管理应用，可以将多邻国导出的记忆卡片批量导入，分类，复习，ai解析的复习及管理工具。

## 原始图片导入
- 利用人工智能将卡片上的中文和英文识别，自动分类(场景，时态)，并记录到本地数据库
- 关于图片识别，我倾向于直接使用视觉大模型就可以了。但鉴于大模型识别的成本问题，我先使用硅基流动的免费模型处理，由于视觉模型价格偏贵，所以我建议先使用硅基流动的ocr模型识别出所有问题，再交给大预言模型提取正确的中英文，再分类，再入库（关于使用哪种方案，使用哪个模型，我们可以讨论）。
- 免费视觉模型：THUDM/GLM-4.1V-9B-Thinking
- 免费ocr模型有：PaddlePaddle/PaddleOCR-VL-1.5，deepseek-ai/DeepSeek-OCR
- 免费大模型有：Qwen/Qwen3-8B，deepseek-ai/DeepSeek-R1-0528-Qwen3-8B
- 复习可以按照多种方式复习，中译汉，汉译英，按场景，按时态，按时间（按导入图片中的文件名中的时间）等
- 除了识别中英文，还要给句子分类，比如场景，时态。
## 数据导出与导入
- 因为是本地数据库，所以要支持数据导出与导入。
## 复习方式
- 复习时也是以做题为主，显示中文，手动输入英文。或者显示英文，手动输入中文。
- 复习时，界面最上方有复习进度。
- 复习时，界面最下方有三个按钮，分别是：上一题，提示，确定。
    - 上一题：返回上一道题目
    - 提示：将正确答案先按词（token）拆分，然后把拆分后的词顺序打乱，显示在界面中，用户点选了某一个词后，该词移动到上方的答题栏中
    - 确定： 系统判定是否正确，这里有两个依据，第一完全和答案一致，立刻判定正确。如果和正确答案不一致，然后由ai判定，ai判定正确，就正确，ai判定错误，就错误，如果做错了，不管正确与否，都要显示出标准的正确答案，回答完之后，界面出现一个ai图标，回答正确时是“AI解读”，从句子的成分，时态，固定搭配，等全方位解读这个句子。回答错误时是“AI解析”，除了AI解读分析的内容，还要重点解释为什么打错了，如果有同类，相似易错的知识一并列出。ai解析，我建议从下方滑出抽屉方式解释，覆盖下方UI界面2/3的范围，，解析界面最下方有输入框，可以对ai的回答继续追问。
    - 点击确定评判后，“确定”按钮变为“下一题”
