# DUORECALL (多邻记)

## 1. 多邻记是什么？
多邻记是一个记忆卡片管理应用，可以将多邻国导出的记忆卡片批量导入，方便用户管理和复习。

## 多邻记的特点
- 利用人工智能将卡片上的中文和英文识别，自动分类，并记录到本地数据库
- 复习可以按照多种方式复习，按场景，按时态，按时间，按短语等
- 复习过程中可以随时调取AI解析句子，做错可以分析错误原因，可以针对中英文展开对话
